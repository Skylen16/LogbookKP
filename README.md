# 📘 Web Logbook Magang KP

## 📌 Deskripsi
Web Logbook Magang KP adalah aplikasi berbasis web sederhana yang dibangun menggunakan HTML dan CSS untuk mencatat serta mendokumentasikan aktivitas harian selama kegiatan Kerja Praktik (KP) atau magang.

Project ini berfokus pada tampilan yang bersih, ringan, dan mudah digunakan tanpa memerlukan backend maupun database, sehingga dapat langsung dijalankan melalui browser.

---

## 🎯 Tujuan Pengembangan
- Menyediakan media pencatatan kegiatan harian selama magang
- Membantu penyusunan laporan Kerja Praktik
- Meningkatkan kerapihan dokumentasi pekerjaan
- Memberikan tampilan logbook yang lebih terstruktur dibanding pencatatan manual

---

## ✨ Fitur Utama
- 📝 Input logbook harian secara manual
- 📅 Penampilan data berdasarkan tanggal kegiatan
- 📊 Tampilan tabel logbook yang rapi dan mudah dibaca
- 🎨 Desain UI sederhana, bersih, dan responsif
- ⚡ Ringan dan cepat karena tanpa backend

---

## 🖥️ Tampilan Aplikasi
Struktur tampilan utama terdiri dari:
- Header (Judul aplikasi)
- Section input kegiatan
- Tabel logbook
- Footer informasi

---

## 🏗️ Teknologi yang Digunakan
Project ini dibuat dengan teknologi dasar web:

- **HTML**
  - Digunakan untuk struktur halaman
- **CSS**
  - Digunakan untuk styling dan layout
  - Mendukung tampilan responsif
